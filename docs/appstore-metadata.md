## App 描述（中文）

歌词投屏是一款专为现场演出、KTV聚会、教会敬拜等场景设计的实时歌词投影工具。

手机搜索歌曲，电脑大屏幕同步显示歌词，让每一位观众都能看到歌词。

【核心功能】
• 三大音乐源：支持QQ音乐、网易云音乐、酷狗音乐搜索歌词
• 实时投屏：歌词通过局域网实时同步到Mac/Windows电脑
• 卡拉OK模式：逐字变色高亮，紧跟演唱节奏
• 13种动画效果：淡入淡出、弹跳、打字机、随机大小等丰富的歌词显示动画
• 翻译歌词：自动获取中英文翻译对照
• 自定义样式：字体、颜色、大小、背景图片完全自定义

【使用方法】
1. Mac或Windows电脑安装LyricsCaster桌面端
2. 手机和电脑连接同一WiFi网络
3. 打开手机App，自动发现并连接电脑
4. 搜索歌曲，选择后歌词自动投影到电脑屏幕
5. 电脑通过HDMI连接投影仪或大屏幕

【适用场景】
• 教会敬拜：大屏幕显示赞美诗歌词
• 现场演出：舞台背景投影歌词
• KTV聚会：家庭投影唱歌
• 音乐教学：歌词同步教学辅助

## App Description (English)

LyricsCaster is a real-time lyrics projection tool designed for live performances, karaoke parties, church worship, and more.

Search songs on your phone, display lyrics on the big screen in real-time. Everyone in the audience can follow along.

【Key Features】
• Three Music Sources: QQ Music, NetEase Cloud Music, Kugou Music
• Real-time Projection: Lyrics sync to Mac/Windows via local network
• Karaoke Mode: Character-by-character highlighting
• 13 Animation Effects: Fade, bounce, typewriter, random size and more
• Translation Support: Auto-fetch bilingual lyrics
• Custom Styles: Font, color, size, background fully customizable

【How to Use】
1. Install LyricsCaster desktop app on Mac or Windows
2. Connect phone and computer to the same WiFi
3. Open the phone app - it auto-discovers the computer
4. Search and select a song - lyrics appear on the computer screen
5. Connect computer to projector or big screen via HDMI

## 审核备注（Review Notes）

This app requires a Mac or Windows computer running the LyricsCaster desktop companion app on the same WiFi network for full functionality.

Testing instructions:
1. The app includes a "Demo Mode" (演示模式) that can be accessed without connecting to a desktop app
2. In Demo Mode, pre-loaded sample lyrics are available to demonstrate the full lyrics display experience
3. For testing the search function: search any Chinese song name (e.g., "月亮代表我的心") and select a result to view lyrics

The app uses local network (MultipeerConnectivity + Bonjour) for device discovery and data transfer between iPhone and Mac/Windows. No internet connection is required for the lyrics projection feature once lyrics are loaded.
